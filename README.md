# BikeRenting – Interactive map using external API

## Project context
This project was developed as part of the **Web Developer** training program by OpenClassrooms.  
The goal was to build a front-end application consuming an external API to display real-time bike rental station data on an interactive map.

The project simulates a real-world scenario where a city bike rental service provides public data through an API.

## Educational objectives
- Consume a REST API using JavaScript
- Handle asynchronous data fetching
- Dynamically update the DOM
- Display geolocated data on an interactive map
- Organize front-end assets in a structured project

## Technologies used
- JavaScript (ES6)
- HTML5 / CSS3
- Leaflet.js
- External REST API
- Git / GitHub
- GitHub Pages (deployment)

## Features
- Display bike rental stations on a map
- Show station details (availability, status)
- Dynamic marker rendering based on API data
- Interactive map navigation
